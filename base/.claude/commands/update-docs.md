# Update Docs

## Steps

1. **Check CLAUDE.md line count** — should be under ~120 lines
2. **Verify docs/INDEX.md accuracy** — all listed docs exist, no missing docs
3. **Sync docs with code** — check recently changed files against their docs
4. **Fix broken links** — verify all doc cross-references work
5. **Remove stale docs** — docs for deleted features should be removed

## Rules

- CLAUDE.md is an index — move details to docs/
- Prefer updating existing docs over creating new ones
- Update docs/INDEX.md when docs change
