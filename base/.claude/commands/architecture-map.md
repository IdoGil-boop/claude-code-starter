# Architecture Map

## Steps

1. **Identify entry points**: main files, route definitions, task registrations
2. **Map key modules**: responsibilities, exports, dependencies
3. **Trace data flow**: request → handler → service → database
4. **Document connections**: which modules depend on which
5. **Output**: structured architecture overview

## Output Format

```markdown
## Entry Points
- path/to/main — API server
- path/to/worker — Background tasks

## Key Modules
- module/ — Responsibility description

## Data Flow
Request → Route → Service → Database
```
